## Quick Start

**Pre-requisites**: 
- Python 3.9+ (3.10 recommended)
- Conda (for environment management)
- Java (for ROBOT ontology tool)

```bash
make conda-env
conda activate mcbo
make install
make demo
```

See the [Installation Guide](https://mcbo.readthedocs.io/en/latest/installation.html) for detailed setup instructions.
