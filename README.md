## Quick Start

```bash
conda create -n mcbo python=3.10 && conda activate mcbo
pip install -r requirements.txt
make install
make demo
```

See the [Installation Guide](https://mcbo.readthedocs.io/en/latest/installation.html) for detailed setup instructions.
